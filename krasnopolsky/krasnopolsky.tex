\begin{abstract}
Genesis Global Group is a collection of vertically integrated companies that spans the biotechnology, healthcare, manufacturing, real estate, and hospitality sectors. During my internship, I developed an IPTV hotel television application using Android Studio and simulated a backend server using MySQL. I connected the application to the backend database, gaining experience in frontend and backend development, and understanding how they interact.
\end{abstract}
